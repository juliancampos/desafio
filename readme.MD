# Solução ao Desafio

## Sistemas
1. analytical-api
2. consumer-api
3. learning-api
4. management-sqs-api

## Base de dados
1. Postgres
2. MongoDB
3. Elasticsearch

## Bibliotecas utilizadas
1. Node.js
2. Npm
3. Express
4. Body-Parser
5. Config
6. js-yaml
7. aws-sdk
8. mongoose
9. sequelize 
10. docker
